Recommendation System & Indexing

  • Data Crawling using jsoup

    • Contents of Java Programming Wikibooks was crawled para wise to provide recommendations based on the Query String.

    Oracle The Java Tutorials web page was crawled based on the tag ("#page-content") to provide recommendations.

  • Lucene Indexing

    • The web data crawled is stored as a seperate text file and used to build the Term Document for the Lucene Recommendation engine.

    Query provided by the user is indexed using lucene and top 10 paragraphs recommended by lucene based on the score are displayed.

  • User Interface

    • The application is built using HTML and javascript.The lucene recommendation engine runs offline and the provides the results

    The 10 queries used in the project can be selected using radio buttons.The Recommendations for the selected post are displayed below the query
